# Task_Manager

Task Manager 

A full-stack task management web application built with Flask.

Features:
- User authentication (login/register)
- Create, edit, delete tasks
- Undo delete functionality
- Due dates for tasks
- Dark mode UI
- Responsive design

Tech Stack:
- Flask
- SQLAlchemy
- SQLite
- Bootstrap
